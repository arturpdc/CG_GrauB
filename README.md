Comandos para funcionar o programa

WASD - Para mover 
Mouse - Controla a camera
Q e E - Controlam a profundidade
F1 - Salva 
F2 - Recarrega
Del e V - Trocam entre visível e invisível
1 ao 9 - Coloca uma cor
